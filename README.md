项目描述
------------

-   项目名：基于微信的记账小程序开发
-   项目版本：2.0
-   项目功能简述：用户使用基于微信的记账小程序，使用“计划”功能配合用户自己输入的每天消费实现记录支出的作用。加上图表和公众号，方便用户的记录，查看和使用
-   代码仓库地址：https://gitee.com/SE-Tally/Tally
-   第一负责人：【Ycl】
